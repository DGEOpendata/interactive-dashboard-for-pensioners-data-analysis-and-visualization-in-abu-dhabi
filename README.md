# Interactive Dashboard for Pensioners Data Analysis and Visualization

This repository provides an interactive web-based dashboard for analyzing and visualizing the 'Pensioners Distribution Data for 2022.' The tool is designed to make the dataset more accessible and actionable for a wide range of users, including policymakers, NGOs, researchers, and businesses.

## Features
- **Dynamic Visualizations**: Explore data through bar charts, line graphs, and pie charts.
- **Data Filters**: Filter data by gender, quarter, and other criteria.
- **Multi-Language Support**: Available in English and Arabic.
- **Downloadable Reports**: Export customized reports for further analysis.

## Requirements
To run this application, ensure you have installed the following:
- Python 3.8+
- Flask
- pandas
- plotly

Install dependencies using:
bash
pip install flask pandas plotly


## How to Run
1. Clone this repository:
bash
git clone <repository_url>

2. Navigate to the repository folder:
bash
cd pensioners-dashboard

3. Place the 'Distribution_of_Pensioners_2022.xlsx' file in the repository folder.
4. Run the application:
bash
python app.py

5. Open your browser and navigate to `http://127.0.0.1:5000`.

## Usage
1. Load the web application in your browser.
2. Use the filters to adjust the data displayed in the visualizations.
3. Interact with the charts to derive meaningful insights.
4. Use the 'Download Report' button to export data for offline analysis.
